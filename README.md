# uma simples api para autenticação com keycloack
rodando na porta ![5065](http://localhost:5065/swagger/index.html)

## para rodar importe os bancos de dados na pasta bacpac
(Importação dos bancos feita de maneira automatica ao rodar o docker)
Rode o docker:
```bash 
docker compose up -d
```

## inicie com 
```bash
dotnet run
```
